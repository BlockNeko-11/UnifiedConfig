# Unified Config

## Installation (Developers)
Latest maven version:
[![](https://jitpack.io/v/BlockNeko-11/UnifiedConfig.svg)](https://jitpack.io/#BlockNeko-11/UnifiedConfig)

### Add repository
In your `build.gradle`:
```gradle
repositories {
    maven {
        url = "https://jitpack.io/"
    }
}
```

### Add dependency
In your `build.gradle`:
```gradle
dependencies {
    implementation "com.github.BlockNeko-11:UnifiedConfig:${rootProject.unified_config_version}"
}
```
